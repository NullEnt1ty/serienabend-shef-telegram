# Serienabend Shef Telegram

Telegram-Bot für Serienabend Shef

## Dev-Setup

### Voraussetzungen

- Python >= 3.9
- pip (i.d.R. mit Python ausgeliefert)
- [Pipenv](https://github.com/pypa/pipenv)

### Installation

1. Abhängigkeiten installieren & Virtual Environment aktivieren:

    ```shell
    pipenv install --dev
    pipenv shell
    ```
